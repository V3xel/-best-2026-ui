Style Guide

Glassmorphism

Frosted glass surfaces, translucent layers, and luminous depth. The style that made blur a design decision.

Layering Depth Translucency 2020 → Now
01

Frosted Surface

Use backdrop-filter: blur() on a semi-transparent container. The blur amount (8–20px) controls frosted intensity.

02

Vivid Background

Glass only reads as glass when the background behind it is colorful. Gradients, images, and neon blobs are perfect backdrops.

03

Subtle Border

A hairline rgba(255,255,255,0.25) border defines the edge of the glass surface without adding weight.

04

Light Fill

Fill the card with rgba(255,255,255,0.10–0.20) in dark contexts, or rgba(255,255,255,0.55) in light.

CSS Recipe

.glass-card {
  background: rgba(255, 255, 255, 0.12);
  backdrop-filter: blur(16px);
  -webkit-backdrop-filter: blur(16px);
  border: 0.5px solid rgba(255, 255, 255, 0.25);
  border-radius: 16px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12);
}

Do

  • Use on vibrant, colorful backgrounds
  • Keep fill opacity low (10–20%)
  • Layer multiple glass panels for depth
  • Use high-contrast text on glass surfaces

Don't

  • Use on plain white or very dark backgrounds
  • Overuse blur — it tanks performance
  • Make the glass so opaque it loses the effect
  • Nest too many glass panels (depth paradox)
