# Best 2026 UI/UX Styles

A curated, open-source showcase of the best UI/UX design styles and trends defining product design in 2026.

**Live site:** [V3exl.github.io/best-2026-ui](https://V3exl.github.io/best-2026-ui)

---

## What's inside

This reference covers 8 major visual systems:

| Style | Description |
|-------|-------------|
| **Glassmorphism** | Frosted glass surfaces, blur backdrops, translucent depth |
| **Neobrutalism** | Thick borders, hard shadows, flat fills — loud and intentional |
| **Bento Grid** | Modular variable-size card layouts |
| **Claymorphism** | Soft 3D inflated shapes with pastel hues |
| **Dark Minimal** | Near-black surfaces, one accent, zero clutter |
| **Retro Futurism** | Terminal grids, scan lines, monospace type |
| **Aurora / Mesh** | Flowing gradient meshes — the AI-era look |
| **Type-First** | Typography as the main visual element |

Each style includes:
- Live interactive demos
- CSS code recipes
- Color palettes
- Do's & don'ts
- Real-world usage context

## Component Library

The site also includes copy-ready components:

- **Buttons** — Primary, Secondary, Outline, Ghost, Neobrutalist, Pill
- **Cards** — Default, Glass, Neo
- **Inputs** — Default, Pill, Brutalist, Underline
- **Badges** — Semantic and stylistic variants

## Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, grid, flexbox, backdrop-filter
- **Vanilla JS** — no frameworks, no dependencies
- **Google Fonts** — Space Grotesk + Inter

## Local development

```bash
git clone https://github.com/V3xel/best-2026-ui.git
cd best-2026-ui

# No build step needed — just open index.html
# Or use a local server:
npx serve .
# or
python3 -m http.server 8080
```

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **"Deploy from a branch"**
4. Select **main** branch, **/ (root)** folder
5. Save — your site goes live in ~60 seconds
